Regex README

What regex does: A regex is a pattern that checks whether text follows a rule. You write the rule once, then test a string against it.

How to build one

Start with the exact text you want. Then replace the changing parts step by step.

Example goal: Match dates like 2026-03-14.

Step 1: Start with the shape: 0000-00-00

Step 2: Replace each number with a digit token: \\d\\d\\d\\d-\\d\\d-\\d\\d

Step 3: Shorten repeated parts with counts: \\d{4}-\\d{2}-\\d{2}

Step 4: Force the whole string to match: ^\\d{4}-\\d{2}-\\d{2}$

Common building blocks

\\d any digit like 0 to 9

\\w word character like a letter, number, or underscore

\\s whitespace like a space or tab

. any single character

[abc] one character from the list

[a-z] one lowercase letter from a to z

[^abc] one character that is not in the list

How repetition works

* zero or more times

+ one or more times

? zero or one time

{3} exactly 3 times

{2,5} between 2 and 5 times

Important symbols

^ start of the string

$ end of the string

(...) capture a part of the match

(?:...) group without capturing

| OR, for example cat|dog

Easy examples

4 digits only: ^\\d{4}$

Date: ^\\d{4}-\\d{2}-\\d{2}$

Email: ^[a-z0-9._%+-]+@[a-z0-9.-]+\\.[a-z]{2,}$

Starts with "cat": ^cat

Contains only lowercase letters: ^[a-z]+$

Flags

i ignore uppercase vs lowercase

g find all matches, not just the first one

m let ^ and $ work line by line

Beginner tip: If your regex feels confusing, build the smallest piece that works first, test it, and only then add more rules.
